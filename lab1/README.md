# Метод Гаусса
Методы ввода данных в программу:  
    • Пользовательский ввод;  
    • Ввод данных из файла;  
    • Генерация случайных матриц.  

Реализовано:  
    • Вычисление определителя;  
    • Вывод треугольной матрицы (включая преобразованный столбец В);  
    • Столбец неизвестных;  
    • Столбец невязок.
